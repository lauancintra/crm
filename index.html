Configure uma vez, use para sempre Siga os 3 passos. Leva menos de 5 minutos.
1
Criar o Google Sheets

Acesse sheets.google.com e crie uma planilha em branco com o nome Villa Veneza — Gestão.

2
Instalar o Apps Script

Abra o arquivo apps-script.js (veio junto com este HTML) num editor de texto, copie todo o conteúdo e cole no Apps Script da planilha.

📋 Passo a passo: Na planilha: Extensões → Apps Script → apague tudo → cole o código → Salve Clique em ▶ Executar → selecione setupSheets → Execute (cria as abas)
3
Publicar e colar a URL
📋 Gerar a URL: Implantar → Nova implantação → Aplicativo da web Executar como: Eu mesmo | Acesso: Qualquer pessoa Implantar → Autorizar → copie a URL gerada
